# Signal-subspace DS (faithful Kanai 2023) — L0 construction sanity (2026-06-20)

Module: `temporal/signal_subspace_ds.py` · Script: `temporal/experiments/ssds_l0.py` · `temporal/outputs/ssds_l0/`
Purpose: validate that I built the lab's actual method correctly (SSA signal subspace → generalized DS with
overlap → learned non-anomalous reference D_N → direction×magnitude change degree), before taking it to real data.

## Construction (ledger §1/§2)
1D two-tone signal; SSA trajectory matrix (w=120 × M=60) per window; signal subspace = top-r=4 left singular
vectors (dominant temporal structure); generalized DS between past (lag τ=16) and present signal subspaces;
reference D_N = principal subspace of the DSs over a separate NORMAL series. Anomaly = phase-continuous
frequency shift of one tone during a segment. Change-point evaluation (present window overlaps the transition).

## Result (15 seeds/type)
| anomaly | a_hat (DS+D_N) | min-angle θ₁ | mean-angle |
|---|---|---|---|
| dominant (dominant tone shifts) | 0.981 | 0.979 | 0.988 |
| secondary (secondary tone shifts; dominant persists) | 0.985 | 0.927 | 0.989 |

## Verdict
- **CONSTRUCTION VALID.** a_hat detects the change at AUC ≈ 0.98; the implementation is faithful and behaves
  as the paper describes.
- **Reproduces the paper's claim:** DS (a_hat) beats the SSA minimum-angle baseline θ₁ on a secondary-component
  change (+0.058), where the dominant component persists so θ₁ stays small.
- **Caveat (the recurring pattern, again):** the *mean*-canonical-angle baseline is equally strong (0.989) on
  this STATIONARY synthetic. So the D_N "model-the-normal" machinery shows no edge over a simple subspace
  statistic here — as expected, because there are no normal *dynamics* to model out. D_N's value can only
  manifest where the normal signal itself changes constantly (seasonality). That is the decisive next test.
- Note: with cumulative-phase integration a frequency-segment becomes a persistent phase offset (≈ a step), so
  the `peak_in_seg` flag (which assumed a transient) reads 0 — the detection is nonetheless correct (a_hat
  fires at the transition and returns to baseline once both windows sit on the new regime).
